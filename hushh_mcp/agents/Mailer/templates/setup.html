Setup Instructions
Google Gemini API
  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy the generated API key
  5. Paste it in the Google Gemini API Key field above
Mailjet API
  1. Visit Mailjet and create an account
  2. Go to API Keys section
  3. Copy your API Key (Public)
  4. Copy your Secret Key (Private)
  5. Paste them in the respective fields above
Important: Keep your API keys secure and never share them publicly. These keys will be stored securely for this session only.
Tip: For production use, consider setting these as environment variables instead of entering them through the web interface.
